# 全球航空简报

## 目标
生成面向航空从业者与1500小时以下飞行员的过去24小时全球航空业简报，输出中文 Markdown。

## 工作流程
1. 明确时间窗口为过去24小时。
2. 优先覆盖地区：东亚、北美、大洋洲；若不足≥10条，再扩展到全球。
3. 采集可验证来源（新闻、航司公告、监管公告、机场/制造商更新、YouTube 等），只用真实URL。
4. 提炼每条的标题、简述、就业影响、地区与来源链接。

## 输出格式（必须满足）
- 中文 Markdown
- ≥10条
- 每条包含：标题 / 简述 / 就业影响 / 地区 / Sources（真实 URL，可含 YouTube）
- 若信息不足，允许相关性稍弱条目补足至 ≥10 条，并明确标注
- 不确定的就业影响请标注为「不确定」
- 每条 Sources 建议 ≥2 个；若仅 1 个，请优先官方或权威行业来源

### 模板
1. 标题：...
   - 简述：...
   - 就业影响：...
   - 地区：...
   - Sources：
     - <真实URL>
     - <真实URL>

## 成功标准
- 满足≥10条且覆盖优先地区优先级规则
- Sources 全为真实 URL（不使用占位链接）
- 就业影响清晰指向飞行员就业或培训/招聘变化
